# Mac OS Homebrew Useful Software
Use Homebrew to install all needed programs for your Mac OS

## Install Homebrew

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## Update Homebrew

```
brew update
brew upgrade
```
